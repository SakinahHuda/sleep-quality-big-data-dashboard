About This Project

This project is an interactive analytical dashboard designed to evaluate sleep quality among patients. Using Big Data analysis, it visualizes key factors affecting sleep, such as lifestyle habits: working hours, caffeine intake, exercise, and stress level. The dashboard aims to provide insights to improve sleep patterns and overall health.

Vision & Mission

Vision

To empower individuals and healthcare professionals with actionable insights to improve sleep quality and health outcomes.

Mission

To create an intuitive, data-driven platform that analyzes sleep patterns and lifestyle habits, providing personalized recommendations for better sleep.

Methodology

The project follows data collection, cleaning, analysis, and visualization using interactive dashboards. Each step is designed to ensure data accuracy and provide actionable insights to improve sleep quality.

Target Audience

This system is designed for the general public to explore sleep and lifestyle patterns through data visualisation. Insights may also support healthcare awareness by helping individuals and stakeholders understand factors affecting sleep quality.

Project Methodology Details

1. 📊 Data Collection: Sleep and lifestyle data were collected from publicly available datasets.

2. 🧹 Data Cleaning: Missing values and inconsistencies were handled to ensure accurate analysis.

3. 📈 Data Analysis: Various statistical and Big Data tools were used to identify patterns affecting sleep quality.

4. 📊 Visualization: Interactive dashboards were created to present insights using charts and tables.

5. 💡 Recommendations: Actionable insights were provided for improving sleep habits based on analyzed data.
