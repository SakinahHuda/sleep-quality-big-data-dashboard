# sleep-quality-big-data-dashboard
Interactive big data analytics dashboard for analyzing lifestyle factors affecting sleep quality using Random Forest and CRISP-DM methodology.

📌 Project Overview

This project develops an interactive analytical dashboard to analyze lifestyle factors influencing sleep quality using big data technologies.

🧠 Methodology
CRISP-DM Framework
Data preprocessing
Random Forest Regression model

🛠 Tools & Technologies
Python
Apache Hadoop
Pandas, Scikit-learn
Power BI

📈 Model Evaluation
RMSE
R²
